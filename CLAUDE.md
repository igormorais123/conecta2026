## Regra de Sincronização Automática

Sempre que qualquer arquivo for criado ou alterado neste projeto:
1. Fazer `git add` dos arquivos modificados
2. Criar um commit com mensagem descritiva em português
3. Fazer `git push origin main` automaticamente
4. **Não perguntar antes de fazer push** - o push é sempre automático após qualquer alteração

## Convenções
- Mensagens de commit em **português brasileiro**
- Projeto de campanha política - todos os textos em pt-BR
- Stack: HTML5 + CSS3 + JavaScript puro (sem frameworks)
